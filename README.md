# Ledger: Secure Browser Injector 🛡️

This extension acts as the zero-knowledge bridge between your Ledger mobile vault and your desktop browser.

## How to Install (Developer Mode)

Because Ledger requires direct, encrypted WebSocket connections to inject credentials, it is currently distributed outside of the Web Store for maximum security and transparency. 

**Follow these steps to deploy the extension:**

1. Go to the [Releases page](../../releases/latest) and download the latest `.zip` file.
2. Extract the `.zip` file to a permanent folder on your computer (e.g., `Documents/Ledger-Extension`).
3. Open Google Chrome (or Edge/Brave) and type `chrome://extensions` into the URL bar.
4. In the top right corner, toggle **Developer mode** to **ON**.
5. In the top left corner, click **Load unpacked**.
6. Select the extracted `Ledger-Extension` folder.

The Ledger Injector is now live and ready to pair with your mobile vault.
